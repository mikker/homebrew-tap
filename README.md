# Mikker Tap

## How do I install these formulae?

`brew install mikker/tap/<formula>`

Or `brew tap mikker/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mikker/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
